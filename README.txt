What does this extension?
	This extension supports the service from www.sharethis.com.
	With that service, you can share your TYPO3-pages with:
	 - facebook
	 - twitter
	 - send a friend an eMail
	 - share with other social networks

What i must do to use this extension?
	1) You must register on www.sharethis.com (after registration, you get a publisher-ID, which you need to configure this plugin)
	2) Install this extension (and configure the publisher-ID inside the extension-manager)
	3) Put plugin on a TYPO3-page
	4) You are finished ;-)
